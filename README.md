# ТЗ №2 Фоминых Кирилл Станиславович ББИ223

### Входные данные:
На вход программы подается файл с числами (от одного до миллиона; числа отделены пробелами)

## Функционал программы
### Функции _max, _min, _mult, _sum.
Программа может посчитать нужную пользователю функцию на выбор:
1. Поиск максимального числа в файле
2. Поиск минимального числа в файле
3. Произведение всех чисел в файле
4. Сумма всех чисел в файле

### Время работы программы
Программа (testTime_tz2.py) считает время, за которое основная программа выполнила все 4 функции для опредленного количество чисел в файле: от 1 до 1000000 с шагом 10000.
